# 메인홈 큰틀 개선 · 심리 설득 설계 (코드 미적용 · 검토용)

> 목표: 고객이 "읽다가 끌려서 상담까지" 가도록, 메인홈을 **의사결정 심리 흐름**으로 재배열·보강.
> 근거: `momentedit-docs` 통점문서(고객 두려움 29개) + 행동경제학/설득심리 원리.
> 톤: 기존 브랜드(절제·예우·본질). 자극적 마케팅 금지 → "조용한 설득".

---

## 0. 핵심 진단 (한 줄)
지금 메인홈은 **"우리가 무엇을 잘하는가(제품)"** 중심이라 정보는 충분하지만,
고객의 **두려움을 먼저 건드려(손실 회피) → 안심시키고(통제감) → 정당화(대비)하는 '감정 설계'가 비어 있다.**
→ 같은 내용도 **순서와 프레이밍**만 바꾸면 훨씬 끌린다.

## 1. 적용할 심리 원리 (메인홈에 쓸 무기)

| 원리 | 의미 | 모먼트에 어떻게 |
|---|---|---|
| **손실 회피** | 얻는 것보다 잃는 고통이 2배 크게 느껴짐 | "추가금·15시간 노동·컨베이어벨트" 고통을 먼저 환기 |
| **대비/앵커링** | 비교 대상이 있어야 가치가 보임 | 일반 웨딩 vs 모먼트 비교표 → 210/280만이 싸게 느껴짐 |
| **인지적 편안함** | 헷갈리면 이탈, 명확하면 신뢰 | 진행 시스템(NOW)·가격 공개·여정 한눈에 |
| **통제감/자율성** | 내가 정한다는 느낌이 애착을 만듦 | 셀프 청첩장·식순·일정 변경 |
| **정직 효과** | 약점을 먼저 말하면 신뢰가 역설적으로 ↑ | "우리가 약속하지 않는 것들" |
| **권위/호감** | 전문가·사람의 얼굴이 안심을 줌 | 디렉터 스토리·계약 투명성 |
| **희소성** | 한정은 가치를 높임 | 하루 3팀 한정 |
| **사회적 증거** | 남들이 했다는 안심 | 후기(운영 후) · 지금은 디렉터 권위로 대체 |
| **피크엔드/내러티브** | 이야기로 기억된다 | "예약→예식→그 후"를 한 흐름 스토리로 |

## 2. 현재 메인홈 심리 진단 (빈 곳)

현재: HERO → ABOUT → VISUAL ARCHIVE → SERVICE → 가격 → ARCHIVE HONOR → LIVE → DIRECTOR → FAQ

| 심리 단계 | 현재 | 진단 |
|---|---|---|
| 주의(가치 끌림) | HERO | 양호 |
| **문제 공감(손실)** | (거의 없음) | ⚠ **가장 큰 공백** — 두려움을 안 건드림 |
| 해결(안심) | ABOUT·SERVICE | 양호하나 '문제' 없이 답부터 나옴 |
| **진행 안심(통제감)** | (없음) | ⚠ 진행 시스템 미홍보 |
| **정당화(대비)** | 가격만 | ⚠ 비교 앵커 없음 |
| **신뢰(정직/증거)** | DIRECTOR | △ 후기·정직한 한계 없음 |
| 행동(CTA) | 산발적 | △ 수렴 약함 |

## 3. 권장 IA 재배열 (설득 퍼널) — 큰틀

> 기존 섹션을 **재배치 + 3개 신설(★)**. 사진/서비스 자산은 그대로 활용.

1. **HERO** — 가치·정체성(끌림). *유지*
2. **★ 문제 공감 "The Reality"** — 한국 결혼식의 피로 3~4개를 조용히 환기(손실 회피). *신설*
   - 예: "새벽 5시부터 자정까지, 평균 15시간." / "30분마다 입장하는 다섯 번째 식." / "견적서에 없던 비용들."
3. **ABOUT "우리의 답"** — 그 순서를 바꾼다(140분·25명·올인원). *유지·문제 뒤로 연결*
4. **SERVICE** — 구체적 무엇(공간·스타일링·시퀀스·청첩장·다이닝). *유지*
5. **★ 진행 시스템 "두 분만의 진행실"** — 예약 후를 어떻게 함께(통제감·명확성). *신설(앞서 기획)*
6. **가격 + ★ 비교표** — 일반 웨딩 vs 모먼트(대비·앵커링). *비교표 신설*
7. **★ 정직한 한계 "We don't promise"** — 못 하는 것 4가지(신뢰 부스터). *신설(작게)*
8. **LIVE** — 디지털 참석 경험. *유지*
9. **DIRECTOR** — 권위·인간미(호감). *유지*
10. **FAQ** — 마지막 반론 처리(부모님 설득·축의금·라이브 등 두려움 직답). *유지·문항 보강*
11. **CTA(상담 신청)** — 명확·반복(행동). *동선 강화*

→ 한 줄 흐름: **두려움 → 답 → 무엇 → 어떻게(진행) → 정당화 → 신뢰 → 경험 → 사람 → 안심 → 행동.**

## 4. 우선순위 (임팩트 × 노력)

| 순위 | 작업 | 심리 효과 | 노력 |
|---|---|---|---|
| **1** | **문제 공감 블록**(HERO 다음 or ABOUT 상단) | 손실 회피 = 전환의 출발 | 낮음(카피 위주) |
| **2** | **진행 시스템 섹션**(기획 완료분) | 통제감·명확성 | 중 |
| **3** | **가격 비교표** | 대비·앵커링 → 가격 저항 ↓ | 중 |
| 4 | **정직한 한계 블록** | 신뢰 급상승 | 낮음 |
| 5 | **FAQ 보강**(부모님·축의금·라이브) | 반론 처리 | 낮음 |
| 6 | **CTA 동선**(섹션 말미 상담 유도 통일) | 행동 | 낮음 |
| 7 | 후기(사회적 증거) | 운영 후 | (대기) |

→ **1·4·5·6은 "카피만"으로 큰 효과**(노력 낮음, 즉시 가능). 2·3은 섹션 신설.

## 5. 카피 방향 샘플 (브랜드 톤 · 자극 X)

- 문제공감: *"가장 빛나야 할 두 사람이, 가장 지치는 하루. 우리는 그 순서가 늘 의아했습니다."*
- 진행시스템: *"예약, 그 다음부터가 진짜 시작입니다."*
- 비교표 헤더: *"같은 하루, 다른 설계."*
- 정직한 한계: *"우리가 약속하지 않는 것들."* (200명·야외베뉴·축의금 회수·부모님 설득)
- CTA: *"두 분의 이야기를 듣는 것부터 시작합니다."*

## 6. 결정 대기
- 큰틀 재배열 방향(섹션 1~11) 동의 여부 / 조정
- 우선순위 1~6 중 **어디부터** 착수할지 (권장: 1 문제공감 → 2 진행시스템 → 3 비교표)
- 톤 최종 점검용 `04_브랜드_키워드_시스템.html` 열람 여부
- 적용은 확정 후 한 번에(현재까지 코드 미반영)
