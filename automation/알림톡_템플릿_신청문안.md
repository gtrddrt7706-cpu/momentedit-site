## 3. 테스트 순서 (GAS 편집기에서)

1. `95_notify` 파일 열고 → `notifySetupCheck()` 실행 → 로그에서 설정 누락 확인
2. `95_notify` 파일 → `notifyTestAdminSms()` 실행 → 디렉터 폰에 문자 오면 SMS 연동 성공
3. 테스트 고객(본인 번호로 만든 코드)으로 `notifyTestCustomerByCode('코드')` → 알림톡(또는 SMS) 수신 확인
4. 이후 실제 흐름(상담 확정 등)에서 자동 발송 — 실행 로그에 `[notify] 발송 OK`가 남음

※ 관리자용 알림(입금신호·변경요청 등)은 템플릿 승인 없이 전부 SMS로 나갑니다.
※ 발송 실패는 본 흐름(계약·입금 처리)을 절대 막지 않고 로그만 남깁니다.
※ 발송 시각: 21시~익일 8시 사이 고객 알림은 자동 보류 후 아침 8시에 발송(야간 방해 금지 설계).
