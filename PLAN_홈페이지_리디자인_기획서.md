# 홈페이지 완성도 향상 기획서 (Redesign Spec)

> 목적: momentedit.kr을 2026년 럭셔리 기준으로 끌어올린다. 두 리서치(클로드 코드 외부 리서치 + 클로드 인컨텍스트 코드 감사)의 채택 결론을 하나의 실행 계획으로 통합한다.
> 작성 기준일: 2026-07-01 · 대상 파일: `index.html`(정적·Vercel 자동배포)

---

## 0. North Star (설계 원칙)

1. **절제(restraint)가 프리미엄을 만든다.** 화려한 효과가 아니라 여백·타이포·사진의 완성도.
2. **진사(#6B2A24)는 히어로의 시그니처 한 순간.** 나머지 섹션은 웜뉴트럴로 조용하게. (코디의 "크림슨 축소" 조언과 사용자의 "진사 히어로" 결정을 이렇게 화해시킨다.)
3. **성능·접근성은 럭셔리의 전제조건**이지 옵션이 아니다. 느리거나 튀면 고급이 아니다.
4. **트렌드는 도구지 정체성이 아니다.** 다 넣지 않고 브랜드에 맞는 2~3개만 장인정신으로.
5. **고객 노출 문구에 전각 줄표(—) 금지** (기존 규칙 준수).

## 1. 현황 진단 (실측 · 2026-07-01)

| 항목 | 현재 상태 | 판정 |
|---|---|---|
| CSS 구조 | 339번 줄 1개 = **33,586자**(minify 1줄) | 🔴 최우선 병목 |
| 진사 사용 범위 | 히어로 모노그램에 한정 | 🟢 이미 좋음 |
| 스크롤 리빌 | IntersectionObserver 5개(JS 구동) | 🟡 CSS 전환 여지 |
| `prefers-reduced-motion` | 히어로 스크립트에 반영 | 🟢 있음(전역 확대 필요) |
| 반응형 타이포 `clamp()` | 6곳 사용 | 🟡 부분 |
| 폰트 로딩 | 구글폰트 CDN + `size-adjust` 폴백 @font-face 3종 + preload + swap | 🟢 양호 / 🟡 **stylesheet 링크 중복 로드** |
| 이미지 | `<img>` 20개 중 **16개 picsum 플레이스홀더**, `<picture>`·AVIF·WebP 전무 | 🔴 실사+포맷 필요 |
| `aspect-ratio`(CLS 방지) | 4곳만 | 🟡 전 이미지 확대 필요 |
| SEO 기본 | title·meta description 존재 | 🟢 있음 |

**핵심 통찰**: 코디 리포트의 "지금 당장" 제안 다수(진사 축소·reduced-motion·clamp·히어로 fetchpriority)는 **이미 구현돼 있다**. 코디는 사이트 소스를 못 봐서 몰랐음. 실제로 남은 큰 레버는 **①CSS 구조화 ②실사 사진 ③이미지 포맷/CLS ④폰트 마감**이다.

## 2. 방향성 — 두 리포트에서 채택 / 보류

### ✅ 채택 (근거 충분·온브리프)
- **웜뉴트럴 팔레트 통일**: 배경 크림 계열(예: #FAF8F1)·텍스트 웜차콜(예: #1C1B1A). 순백 대신 따뜻한 오프화이트 (OBSESD 레퍼런스).
- **풀블리드 사진 + 여백 넉넉한 갤러리 그리드** 아트디렉션 (Aesop·Thyme).
- **조용한 텍스트 링크 CTA** + 비강압적 문의 카피 (OBSESD "non-binding 제안서").
- **AVIF/WebP `<picture>` 캐스케이드**, 전 이미지 `aspect-ratio`로 CLS 제거.
- **CWV 목표치**: LCP ≤ 2.5s · INP ≤ 200ms · CLS ≤ 0.1 (모바일 75퍼센타일).
- **Cormorant Garamond는 디스플레이 전용**, 본문은 Noto Serif/Sans KR.
- **`clamp()`에 반드시 rem 성분**(줌 접근성), line-height 단위 없는 값.

### ⏸ 보류 / 조건부
- **"크림슨을 액센트로 축소"** → 히어로는 진사 유지, **나머지 섹션만** 웜뉴트럴로. (원칙 #2)
- **Pantone 2026 'Cloud Dancer'** 인용 → 팔레트 방향의 참고로만, 마케팅 문구에 직접 인용 전 재확인.
- **페이지 전환·프리로더(OBSESD식)** → 성능 예산 통과 시에만 장기 과제.
- **WebGL/Three.js(Aristide Benoist)** → 오프브리프, 미채택.

### ❌ 미채택
- 과한 패럴랙스·대형 이동(전정기관 이슈·럭셔리 톤과 배치).

## 4. 작업 항목 (임팩트 × 노력 · Phase별)

각 항목: 현황 → 목표 → 방법 → 담당.

### Phase 0 — 기반 (반드시 먼저)
- **P0-1. CSS 구조화** 🔴
  - 현황: 1줄 33,586자 → 모든 디자인 반복이 느리고 실수 잦음.
  - 목표: 읽기 쉬운 소스로 분리(섹션 주석·토큰 상단화). 빌드 없이도 유지보수 가능한 형태.
  - 방법: `<style>` 내부를 논리 섹션(토큰/베이스/레이아웃/컴포넌트/모션/미디어쿼리)으로 재배치. 필요 시 배포 시 minify는 선택.
  - 담당: **나**. (기능·픽셀 무변경 리팩터 후 스크린샷 회귀 확인)

### Phase 1 — 지금 당장 (High Impact / Low Effort)
- **P1-1. 폰트 링크 중복 제거** + preload 1개로 정리. 담당: 나.
- **P1-2. 전 이미지 `aspect-ratio` 부여**(CLS 0 목표). 담당: 나.
- **P1-3. `<picture>` + AVIF/WebP 캐스케이드** 뼈대 적용(플레이스홀더도 포맷만 먼저). 담당: 나.
- **P1-4. `prefers-reduced-motion` 전역 확대**(히어로 밖 섹션까지). 담당: 나.
- **P1-5. 섹션 외 잔여 하드코딩 폰트사이즈 → `clamp()`(rem 포함)**. 담당: 나.
- **P1-6. 웜뉴트럴 팔레트 토큰 정리**(배경/텍스트 계열 통일, 진사는 히어로 한정 유지). 담당: 나 → **너 승인**.

### Phase 2 — 중기 (High Impact / Medium Effort)
- **P2-1. 실사 사진 교체** 🔴 (가장 큰 지렛대). 담당: **너(촬영컷)** → 나(슬롯·보정 톤·그리드).
- **P2-2. 갤러리 아트디렉션**: 풀블리드 히어로 + 여백 그리드, 디테일컷 중심. 담당: 나.
- **P2-3. 스크롤 리빌을 CSS `animation-timeline: view()`로 이전**(+`@supports` 폴백, JS 축소). 담당: 나.
- **P2-4. 문의 퍼널 개선**: 단일 컬럼·즉시 확인 메시지·응답시간 명시·`tel:` 링크. 담당: 나 → 너 카피 승인.
- **P2-5. 신뢰 섹션**: 실명·촬영유형 후기, 프로세스 단계 시각화. 담당: 너(후기 자료) → 나.

### Phase 3 — 장기 (조건부)
- **P3-1. 페이지 전환·프리로더**(성능 예산 통과 전제). 담당: 나.
- **P3-2. 가변폰트 opsz축 + `font-optical-sizing: auto`**. 담당: 나.
- **P3-3. CWV 정기 모니터링 체계**(PageSpeed/CrUX). 담당: 나 안내 → 너 실행.

## 5. 디자인 시스템 명세 (Phase 0에서 확정)

- **컬러**: `--seal:#6B2A24`(히어로 한정) · 배경 크림 · 텍스트 웜차콜 · 힘줄색(hairline) · 골드 액센트 최소. 크림슨 위 크림 반전 텍스트는 **WCAG AA 4.5:1 검증 필수**.
- **타이포 스케일**: 디스플레이=Cormorant(라이트/이탤릭·대사이즈·자간 0.16~0.18em) / 본문=Noto Serif·Sans KR / 행장 45~75자 / 본문 line-height 1.5~1.8 · 헤딩 1.15~1.3.
- **간격 스케일**: 4/8 기반 토큰화, 섹션 간 여백 넉넉히.
- **모션 토큰**: 이징·지속시간·리빌 range를 변수로. 원칙 = opacity 페이드 + 미세 이동, 짧게.

## 6. 완료 기준 (Definition of Done)

- [ ] LCP ≤ 2.5s · INP ≤ 200ms · CLS ≤ 0.1 (모바일)
- [ ] 전 이미지 치수/aspect-ratio 지정, 히어로만 eager+high, 나머지 lazy
- [ ] AVIF·WebP 폴백 제공
- [ ] `prefers-reduced-motion`에서 모든 모션 안전 대체
- [ ] 크림슨 반전 텍스트 대비 AA 통과
- [ ] 고객 문구 전각 줄표 0
- [ ] 픽셀 회귀: 리팩터 전후 스크린샷 diff 확인

## 7. 열린 결정사항 (너가 정할 것)

1. **실사 사진 방향**: 얼굴 노출 가능 여부 → 디테일컷(반지·부케·손·테이블) 중심으로 갈지.
2. **팔레트 확정**: 웜뉴트럴 크림/차콜 톤을 P1-6 시안에서 A/B로 볼지.
3. **범위**: Phase 0~1만 먼저 갈지, 2까지 한 번에 갈지.
