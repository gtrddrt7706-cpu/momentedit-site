## 1. 마이페이지 시스템 완벽 파악 (코드 기준 · 기능 전수)

**여정 9단계**: 신청접수 → 상담확정 → 드레스 시착 → 상담완료(계약요청) → 계약완료(서명) → 입금완료(계약금) → 제작중 → 예식완료 → 결과물 전달

**기능 단위(렌더 19종)와 고객이 하는 일**
| 영역 | 기능 | 고객 행동 |
|---|---|---|
| 전체 | NOW 헤드라인 · 진행바(dots) · 예식 카운트다운 | 지금 내 위치·다음 할 일을 한눈에 |
| 상담 | renderConsult | 일정 선택·변경·수락·취소 |
| 예식일 | renderHold·doHoldEdit·renderChangeWizard | 임시고정(가예약)·날짜 변경(견적/수수료) |
| 시착 | renderFitting | 드레스 시착 동의 |
| 계약 | renderContractRequest·openContractView | 정보 입력→계약서 요청→**채워진 계약서 보고 손글씨 서명** |
| 결제 | renderPayment·Mid·Balance | 계약금·중도금·잔금 입금 알림 + **현금영수증 등록** |
| 내 내역 | renderLedger·ensureLedFab | 결제·영수증·서류·환불예상·임시고정 **상시 확인(우측 고정 버튼)** |
| 제작 | renderProduction + 청첩장·식순·애프터웨딩 위저드 | **직접 제작·예식 전까지 무제한 수정** |
| 결과물 | renderResult | 원본 컷 선택·추가 보정 신청·보정본·영상 수령 |
| 환불 | renderRefundBank | 환불 계좌 셀프 제출 |
| 후기 | 설문 | 만족도·공개 후기 |
| 전반 | 카카오톡 알림 | 단계 전환 시 자동 안내 |

→ 한 줄: **"예약 후 예식까지 전 과정을 한 시스템에서 안내받고, 직접 진행한다."**

## 2. ★ 중복 제외 — '진행' 가치만 남기기 (사용자 지시 2)

메인홈이 **이미 충분히 홍보**하는 것(= 진행실에서 빼기):
- 올인원·숨은비용 없음(ABOUT) · 140분 · 25명 · 디지털참석(LIVE) · 다이닝(SERVICE) · 청첩장 컬렉션/디자인(SERVICE) · 공간·스타일링

진행실이 **유일하게 말할 수 있는 것(중복 아님 · 진행 경험)**:
| 진행 고유 가치 | 왜 중복 아님 |
|---|---|
| ① 예약~예식을 **한 시스템에서 안내받으며 진행** | 메인홈에 '진행 경험' 자체가 전무 |
| ② **NOW**가 다음 할 일을 먼저 안내 | '준비 불안 해소'는 어디에도 없음 |
| ③ 계약·금액·일정·서류를 **언제든 내 손으로 확인**(내 내역) | ABOUT '숨은비용 없음'을 **진행 내내 눈으로 증명**(결과 약속 vs 진행 증명 = 다른 각도) |
| ④ 청첩장·식순을 **직접·무제한 수정**(예식 전까지) | SERVICE는 '결과물 컬렉션'을 홍보 · 여기선 '직접 만드는 진행' |
| ⑤ 예식일 임시고정·변경 등 **진행 편의** | 어른 일정 조율 등 진행 디테일 |

**제외 확정**: 다이닝/애프터웨딩·청첩장 디자인 자랑·가격 자체 → 진행실에서 다루지 않음(중복). 필요 시 ①에 "예식 그 후까지 이어지는 흐름" 한 줄로만 암시.

## 3. 진행실 섹션 카피 v3 (진행 한정 · docs 톤)

- eyebrow: **THE PLANNING COMPANION**
- 타이틀: **예약, 그 다음부터가 진짜 시작입니다**
- 부제: **상담부터 예식까지, 두 분의 진행을 손안에서 함께 안내해 드립니다**
- 여정 4스텝(진행만):
  1. **One Flow · 끊김 없는 흐름** — 상담·시착·계약·입금이 하나의 흐름으로 이어집니다
  2. **Always Next · 다음 할 일** — 지금 하실 일을 'NOW'가 먼저 알려드립니다
  3. **Nothing Hidden · 투명한 내역** — 계약·금액·일정·서류를 언제든 내 손으로 확인합니다
  4. **Yours to Shape · 두 분의 손으로** — 청첩장·식순을 직접, 예식 전까지 마음껏 다듬습니다
- 클로징: **좋은 예식은 당일이 아니라, 준비하는 동안에도 정중합니다.**
- CTA: 없음(마이페이지는 계약 고객 전용)
