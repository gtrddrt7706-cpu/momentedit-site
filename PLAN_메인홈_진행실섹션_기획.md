# 메인홈 신설 섹션 기획안 · "두 분만의 진행실"(마이페이지 시스템 홍보)

> 상태: **기획·카피 다듬는 중 (코드 미적용)**. 확정되면 index.html에 1회 반영.
> 위치(확정): **DIRECTOR(디렉터) 섹션 바로 앞**. 흐름 = 서비스(무엇을) → **시스템(어떻게 함께)** → 디렉터(누가) → FAQ.
> 목적: "촬영·예식·청첩장·다이닝 올인원·숨은비용 없음" 약속을 **증명하는 메커니즘**으로서, 예약 후 예식까지 안내하는 시스템(=웨딩플래너 역할)을 무드에 맞게 자랑.

---

## A. 포지셔닝 (한 줄)

**"예약하는 순간, 두 분만의 디렉터가 손안에서 함께 준비합니다."**

- 경쟁사는 '스드메 패키지'를 팔지만, 모먼트에디트는 **예약 후 예식까지 전 과정을 안내하는 진행 시스템을 기본 제공** = 차별점.
- 톤: "소프트웨어 자랑(차갑게)" 금지 → "손안의 플래너(따뜻하게)". 시스템을 기술이 아니라 **예우·배려**로 번역.

## B. 고객 가치 번역 (기능 → so what)

| 시스템 | 고객 가치 |
|---|---|
| 상담·시착·계약·입금·제작·결과물 한 흐름 | 뭘 언제 할지 헷갈릴 일이 없다 |
| NOW 카드가 다음 할 일 안내 | 옆에서 챙겨주는 플래너 느낌 |
| 계약·금액·일정·서류 '내 내역' 한 곳 | 숨은비용·구두약속 없이 전부 투명 |
| 청첩장·식순 셀프 제작(예식 전까지 무제한 수정) | 내 손으로, 내 속도로 |
| 애프터웨딩 컨시어지(예식 후 식사·가볼곳, 디렉터 검증) | 양가 어른 모실 자리까지 챙겨줌 |
| 24시간 접근·카톡 알림 | 전화 없이 언제든 진행 확인 |

## D. 섹션 구조·비주얼

- 기존 `.sec-label`/`.sec-title`/`.sec-title-sub` 클래스 재사용(무드 일관).
- 4스텝 = 세로 카드 or 2×2 그리드(모바일 1열). 스크롤 진입 시 페이드(기존 reveal 패턴 있으면 재사용).
- (선택) 폰 목업 1컷: LIVE 섹션의 폰 프레임 패턴 재사용해 'NOW 카드' 한 장 노출. **목업 과하면 생략** — 카피·여정만으로도 충분.
- 줄표(—) 금지, '·' 연결, `word-break:keep-all` 유지.
