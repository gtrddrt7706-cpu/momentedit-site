| **01_AI_시스템_마스터.html** | **Couples 시트 17+9 = 26열 마스터 스키마 존재** (A~Q 디지털 공통, R~Z 가족 청첩장 추가). 운영 워크플로우 Phase 1~5 (Miku 정보수집 3~7일 → huijun 자동화 20분) |
| **02_세팅_가이드_v3_2.md** | 상담 절차 3단계(문의서→48h 내 문자→예약금 ₩100,000), 일본어 상담 가능, 부모님 동석 권장 |

### 시점 코드 정의
- **`예약`** = 사전 문의서 시점 (Miku 상담 준비 + 가능 일정 확인 + 우선순위 분류용)
- **`상담`** = 대면 방문 상담 중 대화로 자연스럽게 수집 (폼으로 받지 말 것)
- **`계약`** = 본 계약 후 체크리스트 (예약금 ₩100,000 입금 후 또는 계약서 작성 시)
- **`제작-D`** = 디지털(라이브) 청첩장 생성에 필요 (Couples A~Q)
- **`제작-F`** = 가족(직계) 청첩장 생성에 필요 (Couples R~Z)
- **`당일`** = 예식 당일 송출용 (Vimeo)
- **`운영`** = Miku/Huijun 내부 운영용 (고객한테 직접 입력받지 않음)
