# Part A. 키워드 자동응답 등록 (30개)

## A-1. 등록 방식 안내

### 카카오 관리자센터 접속 경로
```
관리자센터 로그인
  → 채널 선택
  → 좌측 메뉴 [비즈니스 도구]
  → [자동응답 메시지]
  → [신규 등록]
```

### 등록 시 입력 항목
- **키워드**: 1개 자동응답에 여러 키워드 등록 가능 (쉼표로 구분)
- **응답 메시지**: 텍스트 (최대 1,000자)
- **이미지**: 선택 (권장 800×400, 최대 10MB, jpg/png)
- **버튼**: 최대 5개 (웹 링크, 채널 추가, 톡채널 채팅 등)

### 등록 시 주의사항
1. **키워드 중복 방지** — 같은 키워드를 여러 자동응답에 등록하면 충돌 발생
2. **응답 길이 제한** — 1,000자 초과 시 저장 불가 → 초과 시 나누어 등록
3. **버튼 URL** — `https://`로 시작하는 정상 URL만 인식
4. **저장 후 미리보기 확인** — 모바일 화면에서 실제 노출 확인 필수
